# CNN from Scratch

Repositorio de práctica con implementaciones de redes neuronales convolucionales (CNN) usando Python y TensorFlow/Keras.

## Objetivo

Construir y entrenar modelos CNN sobre datasets clásicos, explorando distintas arquitecturas de forma progresiva.

## Stack

- Python 3.10
- TensorFlow / Keras
- NumPy
- Matplotlib
- scikit-learn

## Instalación

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
